# Test Task for Gameloft - Frontend Developer Position

This test task built with Vite 7, React 19, TypeScript, TailwindCSS 4, Eslint and Prettier.

## Tech stack
- [Vite](https://vitejs.dev)
- [ReactJS](https://reactjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwindcss](https://tailwindcss.com)
- [Eslint](https://eslint.org)
- [Prettier](https://prettier.io)
- [React Slick](https://react-slick.neostack.com/) - Smooth carousel functionality with auto-play

## Features
- Product carousel showcasing products
- Shopping cart with discount logic

## Getting Started

Clone the `gameloft-test-task` [repo](https://github.com/slavaxd/gameloft-test-task)

   ```sh
   git clone git@github.com:slavaxd/gameloft-test-task.git

   cd gameloft-test-task
   ```

### Install

Install dependencies.

```bash
pnpm install
```

### Run
Serve with hot reload at <http://localhost:3000>.

```bash
pnpm dev
```

### Lint

```bash
pnpm lint
```

### Typecheck

```bash
pnpm typecheck
```

### Build

```bash
pnpm build
```
